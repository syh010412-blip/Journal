# ------------------------------------------------------------
#  1-1. 상태별 의미
# ------------------------------------------------------------

[상태 의미]
완료 = 할 일을 완수함. 기본 완료 상태
Good = 완료했고 스스로 잘했다고 판단한 것
Try = 완료했지만 시도할 점이 있거나 개선이 필요한 것
Problem = 완료했지만 문제가 발생한 것. 원인 분析이 필요
이후 완료 = 당일에 못했지만 나중에라도 완료한 것. 시간 관리 개선 필요
진행 중 = 시작은 했지만 마무리가 안 된 것. 완료로 전환 필요
못함 = 당일 안에 해야 했지만 하지 못한 것. 실패 원인 분析 필요
취소 = 상대방 사정으로 취소된 것 (내 의지 아님). 실행률에서 제외
시작 전 = 아직 시작하지 않은 것. 당일이 지났다면 상태 업데이트가 누락된 것
핀 = 이후에 다시 참고할 메모/레퍼런스. 할 일이 아니므로 실행률에서 제외
